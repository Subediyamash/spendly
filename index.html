Apple Shortcuts Integration
1
Add to Home Screen
Open this page in Safari → tap the Share button → tap "Add to Home Screen" → name it Spendly → tap Add
2
Create a New Shortcut
Open the Shortcuts app → tap + in the top-right corner to create a new shortcut
3
Set Up the Trigger
Tap Automation → New Automation → scroll to Wallet & Apple Pay → tap Transaction → set it to trigger on any payment
4
Add Three Actions
Action 1 — Ask for Input: Question: "Amount paid? (numbers only)" · Type: Number · Save as amount

Action 2 — Ask for Input: Question: "Merchant name?" · Type: Text · Save as merchant

Action 3 — Open URL: Use the URL below. In Shortcuts, replace the word AMOUNT with your amount variable and MERCHANT with your merchant variable.
5
Save & Configure
Tap Done to save the shortcut. Then toggle "Ask Before Running" to OFF — this makes the automation run silently after each Apple Pay purchase, opening Spendly with the amount and merchant pre-filled.
